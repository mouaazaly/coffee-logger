# Coffee Daily Logger

A full-stack daily logger for a coffee shop SMB. Log daily sales totals and customer counts, view running totals, and track trends over time with a line chart dashboard.

## Requirements
- Node.js (v18 or higher)

## How to run
1. Clone the repo: `git clone https://github.com/mouaazaly/coffee-logger.git`
2. `cd coffee-logger`
3. `npm install`
4. `npm start`
5. Open http://localhost:3000
